User Mode
  • Steps:
    1. Send Login Request
      /userLogin
    2. Send Authentication Code
      /user{token}/authCode
       
  • Official API (bot):
    api.telegram.org/bot{token}/...
  • TDLight API (userbot):
    telegram.rest/user{token}/...
    localhost:XXXX/user{token}/...
     
We only need to authenticate once, the account will stay logged in. We can use the logOut method to log out or simply close the session in our account settings.
